## Building
1. Install [node](https://nodejs.org/en/) & [npm](https://www.npmjs.com/).
2. From the project folder, execute the following command:
```
> npm install
```

3. Run the build command.
```
> gulp build
```

The compiled code will be in the `/dist` folder.
