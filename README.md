## Test

Ensure [Node.js](https://nodejs.org) is installed.

Install [Yarn](https://yarnpkg.com/).
```javascript
> npm install -g yarn
```

Install the dependencies.
```javascript
> yarn
```

Run the tests.
```javascript
> yarn test
```
